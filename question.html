  1. Div: This tag is used to make (divisions) or divide items on a webpage into different parts.

  2. Span: This tag is used to specify and edit a word or certain group of words in a line of text.

  3. A: This tag is used to make a text, image or any object on a webpage a link or directory to another website.

  4. Table: This tag is used to make a representation of a statistical table with intersecting rows and columns.

  5. Form: This tag is used to specify a part of a webpage that is designed to accept input or collect user data and send it to another location for processing usually to grant access to a
    user.

  6. Script: This tag is used to allow javascript code to be inputed directly into the document or add a link to an external javascript file

  7. Title: This tag is used to show the name or identity of a certain webpage on a website.

  8. Br: This tag is used to break a line of text or move a portion of text that is too long to the line below it.

  9. Label: This tag is used to show a text or string that identifies an input field that usually comes below or preceeds it

  10. Link: This tag is used to attach an external file, usually a CSS stylesheet, to a webpage.

  11. Meta: This tag is used to designate the type of characters or text-encoding to be used in a webpage.

  12. Footer: This tag is used to specify all the content that will be found at the bottom of the webpage.

  13. H1: This tag is used to specify the largest heading size

  14. H2: This tag is used to specify the second largest heading size. It is smaller than H1

  15. H3: This tag is used to specify the third ;argest heading size. It is still large but smaller than both H1 & H2

  16. Section: This tag is used to cut out a portion of a webpage to add items into. It functions like the division tag but is quite different.

  17. Ol: This tag is used to create an (ordered) list of specified items. It differs from the unordered list by setting its bullets by default to whole numbers.

  18. Ul: This tag is used to create an (unordered) list of items in a webpage. It differs from the orderd list by using its default bullets as periods although it can be changed.

  19. Input: This tag is used to create an interactable field and allows the webpage to collect a user's data.

  20. Button: This tag is used to create an object on a webpage that can be interracted with. It is usually given a value, a line of text that it displays, and can be clicked to perform an action like acting as a hyperlink(directory to another webpage on the same or different website).

  • Ordered list: This type of list allows list items to be given a more specific bullet type. It uses whole numbers as its bullet type by default.
  • Unordered list: This type of list gives a user the ability to list items with a default bullet style of period. The bullet style can however be changed.
  • Definition list: This type of list gives a brief description, definition or more detail of an item.


  • CSS stands for Cascading Style Sheet. It is used to style or design a webpage as a whole and its constituents individually. It is what determines how beautiful a webpage will be.

    A domain is defined as the name a compuer bears on a network that serves as a direct link to it.
